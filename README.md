# BrainBloom – Online Learning Platform (Frontend)

BrainBloom is a frontend-focused online learning platform designed to provide a clean, structured, and engaging learning experience.

This project follows a **design-to-development workflow**, where all visual and interface elements were designed before implementation.

## What I Designed
- Complete UI/UX design using Figma
- Brand identity and **logo design**
- Color palette and typography
- Page layouts and user flow
- Responsive design system

## Features
- Fully responsive frontend built with HTML, CSS, and JavaScript
- Clean user flow and accessibility-focused design
- Course listing and learning dashboard UI
- Design-first approach with consistent branding

## Tech Stack
- HTML5  
- CSS3  
- JavaScript  
- Figma (UI/UX & Branding)

## Design Process
The entire platform — including **UI/UX, logo, and branding** — was designed from scratch in Figma before development.
Design files and prototype are available in the `/figma` folder.
